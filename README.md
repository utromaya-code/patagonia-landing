# Патагония — край земли · лендинг авторского путешествия

Статический лендинг авторского путешествия по Патагонии (Аргентина + Чили),
**3–15 января 2027**, 12 дней / 11 ночей. Организаторы — Леонид Кутузов и Андрей Баранов.

Стиль и структура — как у leo-japan.ru: фуллскрин-фото, маршрут по локациям,
экспедиционная типографика (Oswald + Golos Text), палитра «ледник / рассвет Фицроя / степь».

## Состав

| Файл | Описание |
|------|----------|
| `index.html` | Главный лендинг — встроенные CSS и JS, без сборки |
| `oferta.html` | Оферта + согласие на обработку персональных данных |
| `img/` | Фото-плейсхолдеры (временные, с Flickr/CC; заменить на реальные) |

Внешних зависимостей нет, кроме Google Fonts и счётчика Яндекс.Метрики.

## Публикация на GitHub Pages

Лендинг лежит в подпапке `patagonia/` репозитория `barinov-courses-landing`
(в корне репозитория — отдельный сайт курсов). После пуша в ветку,
с которой собирается Pages, страница будет доступна по адресу:

```
https://utromaya-code.github.io/barinov-courses-landing/patagonia/
```

Если нужен отдельный сайт под своим доменом — скопируйте папку `patagonia/`
в корень нового репозитория и включите **Settings → Pages → Deploy from a branch → main → /(root)**.
